# ملاحظات التكامل مع تطبيقات الهاتف

- تعتمد المنصة على REST API منظمة حسب الوحدات مثل `/api/auth` و`/api/ministries` و`/api/officials` و`/api/evaluations` و`/api/reports`.
- يفضل لاحقًا اعتماد versioning للـ API قبل فتح الاستهلاك الخارجي.
- نماذج البيانات الحالية صالحة مباشرة لتطبيق Flutter أو React Native.
- يمكن استبدال التخزين المحلي بطبقة S3-compatible دون إعادة تصميم المجال.
- يوصى بإضافة إشعارات لحالات المراجعة والموافقات على المؤشرات الحساسة.
