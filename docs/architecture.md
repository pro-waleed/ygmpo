# معمارية المنصة

تعتمد المنصة على:

- `apps/api`: واجهة خلفية مبنية بـ NestJS مع REST API ونظام صلاحيات وPrisma.
- `apps/web`: تطبيق Next.js App Router بواجهة عربية RTL وصفحات عامة وإدارية.
- `packages/shared`: أنواع مشتركة، بيانات عرض تجريبية، وبعض دوال الحساب.

## مبادئ التصميم

- فصل واضح بين تقييم الوزارة وتقييم الوزير.
- المؤشرات الحساسة لا تُحسب دون دليل ومراجعة.
- كل درجة قابلة للتتبع والتبرير.
- واجهات عامة لعرض النتائج، وواجهات إدارية للإدارة والإدخال والمراجعة.
